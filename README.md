# Getaway

By [Andrea Salkey](www.salkey.com)

## Description
App that searches hotel deals using Expedia's API


## Installation

1. Clone the repo
```
git clone https://github.com/asalkey/getaway_deals.git
```
2. Enter the folder
```
cd getaway_deals
```
3. Install gems
```
bundle install
```
